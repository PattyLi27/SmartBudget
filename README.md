# SMARTBUDGET – Maqueta funcional de landing page

SMARTBUDGET es una maqueta funcional de landing page orientada a personas que desean controlar sus finanzas personales.  
Presenta de forma clara las principales funcionalidades de una futura aplicación web: visualizar ingresos, gastos y metas de ahorro, a través de una estructura modular de secciones. [web:62][web:79]

## Objetivo del proyecto

El objetivo principal es practicar el diseño y desarrollo de una landing page enfocada en un producto digital, definiendo propósito, público objetivo y mensajes clave, como preparación para evolucionar hacia una aplicación web más completa. [web:62]

## Características

- Sección **hero** con nombre del producto y mensaje principal.
- Sección de **características principales** (control de ingresos, gastos y metas de ahorro).
- Sección de **llamada a la acción** para invitar al usuario a empezar a organizar su presupuesto.
- Diseño modular, pensado para futuras iteraciones del producto.
- Capturas de pantalla almacenadas en la carpeta `screenshots/` para documentar la interfaz. [page:2]

## Tecnologías utilizadas

- HTML5 – estructura del contenido.
- SCSS / CSS3 – estilos, maquetado y adaptación responsive. [page:2]
- JavaScript – comportamiento básico de la maqueta (si aplica).

## Estructura del proyecto

- `index.html` – archivo principal de la landing page.
- `sass/` – archivos fuente de estilos en SCSS.
- `css/` – hoja(s) de estilo compiladas.
- `screenshots/` – capturas de la interfaz en escritorio, tablet y móvil.
- `Justificación del proyecto SMARTBUDGET.pdf` – documento de justificación del proyecto. [page:2]

## Cómo visualizar la maqueta

1. Clonar este repositorio:

```bash
git clone https://github.com/PattyLi27/SmartBudget.git
```

2. Abrir la carpeta del proyecto.
3. Abrir el archivo `index.html` en el navegador para ver la landing. [web:160]
